🔧 亚马逊客服·日常工作流程

  • Amazon 客服工作内容与流程
    • 1. 工作目标与计划
      • 提升客户满意度与回复效率
      • 降低负面反馈与退货率
      • 标准化常见问题的答复模板
      • 熟悉平台（Amazon/TEMU）的政策与流程
      • 建立客户沟通档案，记录售前/售后
      • 每周更新问题库与模板
    • 2. 每日客服工作内容
      • 2.1 站内信回复
        • 缺货 → 建议关闭订单
        • 配件缺失 → 比对清单说明
        • 物流延迟 → 建议等待或关闭
        • 签收争议 → 建议报警
        • 商品误买 → 推荐其他型号
        • 损坏/安装异常 → 要求照片
        • 尺寸/参数问题 → 查说明书
        • 兼容性问题 → 提供接口尺寸
      • 2.2 沟通技巧与模板使用
        • 语气专业且自然
        • 使用标准模板
        • 主动解释平台规则
        • 纠正机器人错误回答
      • 2.3 多渠道客服（TEMU）
        • 禁止说 cancel，改为关闭订单
        • 注意人工介入方式
        • 避免说“没货”，改为暂时无法发货
    • 3. 常见问题与处理流程
      • 3.1 缺货
        • 提醒买家无库存
        • 建议关闭订单
        • 推荐替代型号
      • 3.2 签收争议
        • 查物流状态
        • 解释照片送达制
        • 建议报警或联系快递
      • 3.3 商品描述争议
        • 核查页面内容
        • 询问产品详细资料
        • 安抚买家情绪
      • 3.4 兼容性问题
        • 询问接口尺寸
        • 提供产品适配说明
        • 推荐转接配件
    • 4. 差评与 Feedback 处理
      • 4.1 收集信息
        • 获取订单号、邮箱、问题描述
        • 截图差评内容存档
      • 4.2 处理流程
        • 引导站外沟通（邮箱截图/附件）
        • 提供补发配件或部分退款方案
        • 避免使用“差评”等敏感词
        • 委婉鼓励买家移除 Feedback
